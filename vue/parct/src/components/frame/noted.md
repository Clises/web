

Mint-ui(直接全局使用就行) 
1.安装
npm i mint-ui -S

2.引入mint Ui 的 css 插件
  import MintUI from 'mint-ui'
  import 'mint-ui/lib/style.css'
  Vue.use(MintUI)

3. 直接使用
