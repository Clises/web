1.CSS透明度颜色设置问题
 Android部分不支持 hex写法，推荐用rgba的写法
   #0000009c --> rgba(0, 0, 0, 0.61)
2.flex对低版本的ios和Android的支持问题
使用postcss的autoprefixer插件，自动添加浏览器内核前缀，
并且增加更低浏览器版本的配置，自动添加flex老版本的属性和写法

autoprefixer({
    browsers: [
        'iOS >= 6',     // 特殊处理支持低版本IOS
        'Safari >= 6',  // 特殊处理支持低版本Safari
    ],
}),
